QS5 qPCR example

This directory contains sample Applied Biosystems QuantStudio 5 qPCR files and a reference compiled output.

Folder structure
qs5/
├── input/
├── expected_output/
└── README.md
input/

Contains the QS5 Excel files processed by the example script.

Supported formats:

.xlsx
.xls

Each workbook must contain a worksheet named:

Results
expected_output/

Contains the reference output generated by the QS5 compilation script.

Run the example

Run the following command from the repository root:

python scripts\compile_qs5.py `
    --input "examples\qpcr\qs5\input" `
    --output "examples\qpcr\qs5\expected_output\qs5_compiled.xlsx"

The same command can be entered on one line:

python scripts\compile_qs5.py --input "examples\qpcr\qs5\input" --output "examples\qpcr\qs5\expected_output\qs5_compiled.xlsx"
Processing performed

The QS5 reader:

reads the Results worksheet;
skips the first 22 worksheet rows;
removes completely blank rows;
removes rows containing information only in columns A or B;
extracts the required result columns;
converts text values of Undetermined in the Ct column to 40;
adds the source filename;
extracts metadata from the filename;
combines all valid QS5 files into one Excel workbook.

Filename convention

Input filenames should follow this structure:

YYYYMMDD_TestMethod_PlateNumber_Instrument_Operator.ext

Example:

20230818_test_p14_qs5-instr1_op1.xlsx

The filename components are used to populate:

Date
Test method
Plate number
Instrument
operator
Data status

All example files should be anonymised, synthetic, or approved for public release.
