7500 Fast qPCR example



This directory contains sample Applied Biosystems 7500 Fast qPCR files and a reference compiled output.



Folder structure

fast7500/

├── input/

├── expected\_output/

└── README.md

input/



Contains the 7500 Fast Excel files processed by the example script.



Supported formats:



.xls

.xlsx

.xlsm



Each workbook must contain a worksheet named:



Results

expected\_output/



Contains the reference output generated by the 7500 Fast compilation script.



Run the example



Run the following command from the repository root:



python scripts\\compile\_fast7500.py `

&#x20;   --input "examples\\qpcr\\fast7500\\input" `

&#x20;   --output "examples\\qpcr\\fast7500\\expected\_output\\fast7500\_compiled.xlsx"



The same command can be entered on one line:



python scripts\\compile\_fast7500.py --input "examples\\qpcr\\fast7500\\input" --output "examples\\qpcr\\fast7500\\expected\_output\\fast7500\_compiled.xlsx"

Processing performed



The 7500 Fast reader:



reads the Results worksheet;

skips the first 15 worksheet rows;

removes completely blank rows;

removes rows where only column A contains information and columns B to K are empty;

extracts the required result columns;

removes the unused original column F;

maps the raw Ct column to the output column named Ct;

converts text values of Undetermined in the Ct column to 40;

adds the source filename;

extracts metadata from the filename;

combines all valid 7500 Fast files into one Excel workbook.

Output columns



The compiled output contains:



Well

Sample Name

Detector

Task

Reporter

Ct

Automatic Baseline

Baseline Start

Baseline End

Call

Call Assessment

Source File

Date

Test method

Plate number

Instrument

Operator

Filename convention



Input filenames should follow this structure:



YYYYMMDD\_TestMethod\_PlateNumber\_Instrument\_Operator.ext



Example:



20231214\_test\_p01\_7500fast-instr1\_op4\_data.xls



Only the first five underscore-separated filename components are used to populate:



Date

Test method

Plate number

Instrument

Operator



Additional filename components, such as data, are not added as separate metadata columns.



Requirements



Reading legacy .xls files requires:



xlrd>=2.0.1



Install or upgrade it with:



python -m pip install --upgrade "xlrd>=2.0.1"

Data status



All example files should be anonymised, synthetic, or approved for public release.



Do not upload confidential, clinical, personal, proprietary, or identifiable laboratory data.
