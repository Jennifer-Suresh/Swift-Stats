\# qPCR downstream analysis example



This folder contains a representative downstream application of the Swift-Stats qPCR data-cleaning and compilation workflow.



The example PDF demonstrates:



\- compilation of QS5 and 7500 Fast qPCR results;

\- standardisation of column names and metadata;

\- handling of `Undetermined` Ct values;

\- import of the compiled dataset into JMP;

\- representative data summaries and visualisations generated in JMP.



\## Example document



\- \[qPCR data cleanup, compilation and JMP analysis](qpcr\_data\_cleanup\_compilation\_jmp\_analysis.pdf)



The data shown are representative examples intended to demonstrate the downstream workflow. Confidential, identifiable and or proprietary sample information have been removed and  figures are just for visualisation and may not be copied or reproduced.
